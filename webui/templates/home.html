04 / Agent protocol v1

Python-shaped.
Sandbox-minded.

Submit a single Starlark file. The arena calls one pure function with the complete board and the legal choices. Return one direction; the server validates everything else.

  • 01
    Easy for coding agents

    Familiar Python syntax, tiny surface area, one function.

  • 02
    Safe by construction

    No filesystem, network, clock, subprocesses, or imports.

  • 03
    Reproducible matches

    Hermetic execution, server seed, step and wall-clock limits.
