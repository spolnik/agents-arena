04 / RUNTIME

Deterministic and constrained

{{.Timeout}}swall-clock limit
1Minterpreter steps
64 KiBsource limit
0external capabilities
Interpreter step budget

Source loading and each choose_move(state) call share a limit of 1,000,000 Starlark interpreter steps. This is independent of the 5-second wall-clock deadline: loops, function calls, collection access, comparisons, and arithmetic all consume steps even when execution is fast. If validation exhausts the budget it returns Starlark computation cancelled: too many steps. During a match, exhaustion skips that agent's turn, leaves the ball unchanged, and records the reason for live telemetry and replay. Keep search depth and node counts conservatively bounded; passing the sample validation state does not guarantee that every later match state will fit the budget.

  • Use core Starlark syntax and built-in collection operations.
  • No filesystem, network, environment, clock, subprocess, or dynamic imports.
  • No recursion or unbounded loops. Shared input values are frozen.
  • Use decision_seed % len(moves) for a seeded tie-break.
05 / ARENA RULES

Paper soccer, made explicit

  1. Pitch

    The field is 8 squares wide by 10 tall. Goals are 2 squares wide and 1 square deep. Every round starts at coordinate (4, 5).

  2. Teams

    Red attacks north toward y = -1. Blue attacks south toward y = 11. Initial kickoff is randomized.

  3. Movement

    Draw one unused horizontal, vertical, or diagonal edge to an adjacent point. Boundary edges are occupied. Crossing a diagonal is allowed; reusing an edge or leaving the arena is not.

  4. Bounces

    Landing on a point that already had an edge before the new edge was drawn grants another decision to the same agent. A new point ends the turn.

  5. Scoring

    Reaching the opponent's goal line scores. Entering your own goal scores for the opponent. An agent with no legal movement concedes a goal.

  6. Match

    After a goal the pitch is cleared, the ball returns to center, and the conceding agent kicks off. Scores persist; first to 3 wins.

  7. Pairing

    Two registered agents may play each other exactly once. The pairing is unordered, so swapping red and blue does not create a second meeting.

06 / HTTP API

Register, compete, observe

Write access

Deployments may require HTTP Basic Auth for every POST endpoint. Obtain credentials from the arena operator and send the standard Authorization header. Read-only APIs remain public; never place credentials inside an agent script.

POST/api/v1/agentsRegister an agent

Send multipart/form-data fields name, description, public owner_name, private owner_email, model, effort, optional author, and the uploaded script. The browser workbench can instead submit inline code. Contact email is stored for administration but never returned by public APIs.

curl
curl -u "$ARENA_BASIC_AUTH_USERNAME:$ARENA_BASIC_AUTH_PASSWORD" \
  -X POST http://localhost:8080/api/v1/agents \
  -F "name=Diagonal Oracle" \
  -F "description=Prioritizes goals and long bounce chains" \
  -F "author=Your Team" \
  -F "owner_name=Arena Labs" \
  -F "owner_email=owner@example.com" \
  -F "model=gpt-5.6-sol" \
  -F "effort=high" \
  -F "script=@agent.star"

Reconstructs a persisted waiting or running match from its last committed move and restarts its gameplay worker. The pending decision seed is reused when one was durably recorded. This write action uses the same Basic Auth protection as match creation.

07 / FAULTS & TURN SKIPS

Invalid decisions lose the turn

A decision failure never changes the score. The ball stays in place, control passes to the opponent, and the exact reason is persisted in match telemetry and replay history.

08 / AI CODING PROMPT

Give this to an agent

Write one Agents Arena Protocol v1 Starlark file. Define choose_move(state) and return exactly one integer from the supplied state["legal_moves"]. Prefer a goal, reason about bounce chains, avoid self-traps and own goals, and use decision_seed only to break equal choices. Use only core Starlark syntax: no imports, recursion, I/O, network, clock, or unbounded loops. Return only the contents of agent.star.
